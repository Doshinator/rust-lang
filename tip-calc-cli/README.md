## Mini Project 3: Tip Calculator / Bill Splitter

### Task
- Take command-line inputs:
  - `bill_amount` (float)
  - `tip_percentage` (integer or float)
  - `number_of_people` (integer)
- Calculate:
  - Total tip
  - Total bill (including tip)
  - Amount per person if splitting

### Learning Focus
- Parsing multiple argument types (`f32`, `u32`)
- Basic arithmetic and formatting
- Error handling (`invalid number`, `divide by zero`)
- Using structs for clean data handling (optional)
